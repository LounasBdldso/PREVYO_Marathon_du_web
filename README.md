## 📌 About the Project

**PREVYO Analytics** is an open-source data analysis project developed during a 48-hour hackathon (Marathon du Web) in collaboration with EMVISTA. The goal of this project is to analyze a complex Knowledge Base PREVYO, generated by NLP algorithms from raw text, to extract actionable insights.

Instead of extracting the text ourselves, we built a complete data pipeline to audit the existing graph, detect structural anomalies, find semantic similarities between events, and visualize the entire network in an interactive dashboard.

-----

## ⭐️ Features

  - 🧹 **JSON to Graph Parsing**: Automated flattening of nested JSON data into scalable formats.
  - 🗄️ **ArangoDB Integration**: Graph-oriented database setup for sub-graph visualization and exploration.
  - 📊 **Exploratory Data Analysis (EDA)**: Treemaps, Heatmaps, and temporal analysis of event distributions.
  - 🚨 **Anomaly Detection**: Unsupervised machine learning to identify isolated nodes and structural outliers.
  - 🔗 **Similarity & Clustering**: Grouping similar events and detecting quasi-duplicates using advanced embeddings.
  - 📈 **Interactive Dashboard**: A user-friendly Streamlit interface to explore KPIs, clusters, and alerts in real-time.

## 🚀 Installation & Setup

### ⚙️ Data Preparation & Environment

```bash
# Clone the repository
git clone https://github.com/LounasBdldso/PREVYO_Marathon_du_web.git
cd PREVYO_Marathon_du_web

# Create and activate a virtual environment
python -m venv venv
source venv/bin/activate  # On Windows use `venv\Scripts\activate`

# Install dependencies
pip install -r requirements.txt

# Run the JSON parsing script to generate the tabular data
python scripts/parse_prevyo.py --input data/raw_events.json
```

> ⚠️ Ensure you have ArangoDB installed and running locally if you wish to use the graph database visualization features.

-----

### 🎮 Client Launch (Streamlit)

Once the data is pre-processed, you can launch the interactive dashboard:

```bash
# Navigate to the app directory
cd app

# Start the Streamlit server
streamlit run dashboard.py
```

## 🌐 Dashboard Version

The interactive dashboard allows users to navigate through the 4 main views of our project:

1.  **Global KPIs**: Health and stats of the knowledge base.
2.  **Anomaly Explorer**: Filterable list of suspicious events with explicability scores.
3.  **Similarity Map**: 2D UMAP projection of event clusters.
4.  **Graph Viewer**: Raw data and JSON exploration.
